# Girl Scout Troop Web App

## What This Project Is

A website for **Troop 723** (Bayamon, PR — Region 3) to give families one place for events, activities, and contact info. The project is also used to practice and improve programming skills.

## Goals

- **Build a structured plan** — Define and follow a clear development plan.
- **Develop it yourself** — Implement the app independently as much as possible.

## Current plan

- **Phase 1:** Central information hub — one website everyone can use (events, activities, contact info). Complete this first.
- **Phase 2 (later):** Parent login and scout profiles (patches earned, projects in progress).

## Project status

Phase 1 requirements are defined. Target: information hub live next month. Stack: plain HTML/CSS/JS; bilingual (EN/ES); GitHub Pages. Content maintained by Carlos De Jesus.
